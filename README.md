# Customer Churn Prediction using Machine Learning

## Project Description
This project focuses on predicting customer churn using predictive analytics
and machine learning techniques. The goal is to identify customers who are
likely to stop using a service and help businesses improve retention strategies.

## Dataset
The dataset contains customer demographic details, service usage information,
billing data, and churn labels. The dataset was collected from Kaggle.

## Machine Learning Models Used
- Logistic Regression
- Decision Tree Classifier
- One additional classification model

## Tools and Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Results
Logistic Regression provided stable and reliable performance for predicting
customer churn based on evaluation metrics.
